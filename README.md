# TerraLine

Turn-based strategic grid game built with Godot Engine.

## Current Features
- Dynamic grid generation
- Turn-based system
- Card system (+2, +3, Tetromino J & L)
- Automatic card draw every 5 turns

## Architecture
- GameManager → Game rules & turn system
- Grid → Board logic & rendering
- UI → Interface display
